== Description ==

Provides functions that assist in converting your categories/archives to indexes rather than displaying the actual posts.

This plugin will _probably_ work with WordPress 2.1 as well.


== Installation ==

1) Download the plugin.
2) Upload the plugin to your wp-content/plugins directory.
3) Activate the plugin.

You can find detailed instructions on using the plugin at the plugin's web page:

http://www.learncpp.com/wordpress/category_archive_indexing_plugin/
